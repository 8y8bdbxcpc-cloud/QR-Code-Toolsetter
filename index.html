QR-Code-Toolsetter

Stand-alone Rechner für SolidPro und Drehwerkzeuge. Für Sternrevolver und Scheibenrevolver mit editierbaren Basiswerten, Voreinstellwerten und frei einstellbarer Lage. Alles läuft lokal im Browser und bleibt auf dem Gerät gespeichert.

SolidPro Schnellrechner

Für den QR-Code direkt am Werkzeug. Standardmäßig wird der Mittenversatz von 26,5 mm verwendet. Das eingegebene Werkzeugmaß wird ab Adapter / ab Mittenversatz erfasst.
Ergebnis
SolidPro Position
154.000 mm
26.5 + 127.5 = 154.0
Dieser Bereich ist absichtlich einfach gehalten: Position = Mittenversatz + Werkzeugmaß ab Adapter.

Drehwerkzeug / Voreinstellung

Die Sternrevolver-Logik ist generisch aufgebaut: Werkzeugwerte aus der Voreinstellung werden um den wirksamen Winkel gedreht. Das ist bewusst editierbar, weil Vorzeichen, Nullwinkel und Lage je nach Maschine variieren.
Maschine / Layout

Optional: SolidPro aus Drehwerkzeugwert ergänzen

Wenn du vom gleichen QR-Link aus zusätzlich die SolidPro-Position brauchst, kannst du hier das Werkzeugmaß ab Adapter direkt in den Schnellrechner übernehmen.

Wichtige Hinweise

  • Die Seite ist bewusst stand-alone. Du kannst sie direkt als index.html in GitHub Pages ablegen.
  • Für Sternrevolver wird der lokale Voreinstellvektor rechnerisch um den wirksamen Winkel gedreht. Das ist eine saubere, aber generische Logik. Exakte Nullwinkel und Vorzeichen müssen zur Maschine passen.
  • Für Scheibenrevolver wird standardmäßig keine stationsabhängige Drehung angewendet. Die Werkzeuglage kann aber zusätzlich über den Winkel vorgegeben werden.
  • Hauptspindel und Gegenspindel beeinflussen hier nur das Z-Vorzeichen. Falls deine Maschine anders reagiert, kannst du die Basis- oder Zusatzwerte anpassen.
  • Wenn der Bediener X in Durchmesser eingibt bzw. braucht, wird das Ergebnis automatisch entsprechend ausgegeben.
  • Vor dem produktiven Einsatz mindestens einmal mit einem bekannten Referenzwerkzeug gegenrechnen.
